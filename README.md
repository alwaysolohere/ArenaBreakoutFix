# ArenaBreakoutFix
Fix Minimum Requirements error for Arena Breakout Infinite.

# How to use
1. Drop ArenaBreakoutFix.exe to <Your_Path>\Arena Breakout Infinite\launcher
2. Start ArenaBreakoutFix.exe.
